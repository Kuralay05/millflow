# Описание модулей MillFlow

## 1. Панель управления

Показывает ключевые показатели предприятия: объем принятого зерна, объем произведенной муки, складские остатки, количество отгрузок, позиции с низким остатком и последнюю активность.

## 2. Прием зерна

Позволяет регистрировать поступающие партии зерна от поставщиков, хранить сведения о типе зерна, массе, влажности, дате поступления и статусе качества.

## 3. Производство

Используется для учета производственных партий. В модуле фиксируются связанные партии зерна, дата производства, объем произведенной муки, количество отходов и статус выполнения.

## 4. Склад

Предназначен для контроля остатков продукции и сырья. Система показывает количество на складе, место хранения, тип продукции и автоматически выделяет позиции с низким остатком.

## 5. Отгрузки

Содержит сведения об отправках продукции клиентам: номер отгрузки, клиент, партия, количество, дата и текущий статус.

## 6. Отчеты

Формирует сводную информацию по приему зерна, производству, складу и отгрузкам. Подходит для скриншотов, печати и использования в презентации.

## 7. О проекте

Содержит формальное описание назначения системы, задач, ролей пользователей и ожидаемой пользы для предприятия.

## 8. Журнал действий

Доступен только администратору. Фиксирует действия пользователей по созданию, изменению и удалению записей.

## Роли и права доступа

- `admin` — полный доступ ко всем разделам
- `operator` — панель управления, прием зерна, производство, отчеты
- `warehouse_manager` — панель управления, склад, отгрузки, отчеты

## Бизнес-ценность для мукомольного предприятия

- снижение ручного труда при учете операций
- повышение прозрачности движения зерна и продукции
- улучшение контроля складских остатков
- ускорение подготовки отчетности
- уменьшение вероятности ошибок при работе с данными
